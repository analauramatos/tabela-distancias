### Descrição

O projeto consiste na composição de uma tabela que conta com capitais brasileiras e a distancia entre essas capitais. As capitais constantes nesse projeto são: 

* Brasília 
* Campo grande 
* Cuiabá
* Curitiba
* Florianópolis
* Fortaleza
* Goiania
* João pessoa
